# Consequence Boundary

Execution-boundary infrastructure for consequence-bearing autonomous systems.

Consequence Boundary evaluates whether autonomous or AI-assisted execution should remain permissible before irreversible consequence formation occurs.

## Core Principle

```text
Probabilistic cognition may propose actions.

Governance determines whether execution remains admissible.
```

## Public Repository Scope

This repository contains:
- public-facing architecture
- sanitized governance examples
- integration references
- documentation
- demo evaluation flows

This repository intentionally excludes:
- private operational infrastructure
- authenticated execution pipelines
- production secrets or credentials
- enterprise telemetry infrastructure
